Magic: The Gathering Card Data Reports

Welcome to the Magic: The Gathering card data reports. This page provides access to various aggregated data reports about Magic: The Gathering cards.

Use the navigation menu on the right to explore different reports:

  • count_cards_by_name: Count of unique cards by name
  • count_finishes_by_name: Count of card finishes by name
  • count_cards_by_set_name: Count of cards by set and name
  • count_finishes_by_set_name: Count of card finishes by set and name
  • count_card_illustrations_by_set: Count of unique card illustrations by set
  • max_collector_number_by_set: Maximum collector number by set
  • maximal_printed_types: Cards with maximal printed types
  • promo_types_by_name: Promo types by card name
  • first_card_by_power_toughness: First card for each unique power/toughness combination
  • foil_types_by_name: Foil types by card name
  • supercycle_completion_time: Time to complete supercycles
  • maximal_types_with_effects: Cards with maximal types, considering global effects
  • first_card_by_generalized_mana_cost: First card for each generalized mana cost

The data used in these reports is based on the Scryfall database as of 2024-11-15 09:02:28.
